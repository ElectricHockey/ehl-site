Import Historical Data

Import past seasons, game results and player stats from mystatsonline.com or any compatible JSON file.
Run the scraper script first: node scripts/scrape-mystatsonline.js 73879 then upload the generated import-data.json below.

Upload JSON File

Maximum file size: 10 MB. Only JSON files are accepted.

Or Paste JSON

📋 JSON Format Reference 
{
  "seasons": [
    {
      "name": "Season 1",              // required
      "league_type": "threes",         // "threes" | "sixes" | "" (optional)
      "games": [                       // optional — game results
        {
          "date": "2022-01-15",        // YYYY-MM-DD
          "home_team": "Team A",
          "away_team": "Team B",
          "home_score": 5,
          "away_score": 3,
          "is_overtime": false         // optional
        }
      ],
      "player_stats": [                // optional — season aggregate stats
        {
          "team": "Team A",
          "player_name": "PlayerX",
          "position": "C",             // "G" for goalies
          "games_played": 10,
          "goals": 5,
          "assists": 8,
          "plus_minus": 3,
          "pim": 4,
          "shots": 30,
          "pp_goals": 1,
          "sh_goals": 0,
          "gwg": 1,
          // Goalie-only (optional):
          "saves": 180,
          "save_pct": 0.912,
          "goals_against": 17,
          "goalie_wins": 7,
          "goalie_losses": 3,
          "shutouts": 1,
          "gaa": 2.15
        }
      ]
    }
  ]
}

Import from Excel Schedule

Upload an Excel schedule exported from mystatsonline.com.
The server will read each game row and automatically fetch per-game player stats directly from the mystatsonline game-score page using the IDGame column.
Expected columns: Date/Time · Home team · Home score · OT · Away score · Away team · Location · Status · IDGame
